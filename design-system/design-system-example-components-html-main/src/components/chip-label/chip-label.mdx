## 仕様

### バリエーション

| 属性 | 値 | 説明 |
| --- | --- | --- |
| `data-style` | `text`, `outline`, `filled-outline`, `fill` | チップラベルのスタイル |
| `data-color` | `gray`, `blue`, `light-blue`, `cyan`, `green`, `lime`, `yellow`, `orange`, `red`, `magenta`, `purple` | チップラベルの色 |

## 使い方

### アイコン付きチップラベル

アイコンを表示する場合は、`<svg>`要素に`.dads-chip-label__icon`クラスを付与します。アイコンは装飾的なため`aria-hidden="true"`を設定してください。

```html
<span class="dads-chip-label" data-style="text" data-color="gray">
  <svg class="dads-chip-label__icon" width="24" height="24" viewBox="0 0 24 24" fill="currentcolor" aria-hidden="true">
    <!-- アイコンのパス -->
  </svg>
  ラベル
</span>
```

### アイコンなしチップラベル

アイコンが不要な場合は`<svg>`要素を省略します。

```html
<span class="dads-chip-label" data-style="text" data-color="gray">
  ラベル
</span>
```

### 任意の色を使う

所定のCSSカスタムプロパティを使用することで、チップラベルに任意の色を指定できます。

```html
<span class="dads-chip-label" data-style="filled-outline" style="
  --_non-text: #854f91;
  --_bg: #f4dff9;
  --_text-dark: #72447d;
">
  <svg class="dads-chip-label__icon" width="24" height="24" viewBox="0 0 24 24" fill="currentcolor" aria-hidden="true">
    <path d="M4.6 20.5c-.5-.1-1-.6-1.1-1l16-16c.5.1.9.6 1 1l-16 16Zm-1.1-6.4v-2L12 3.4h2.1L3.5 14.1Zm0-7.4V5.3c0-1 .8-1.8 1.8-1.8h1.4L3.5 6.7Zm13.8 13.8 3.2-3.2v1.4c0 1-.8 1.8-1.8 1.8h-1.4Zm-7.4 0L20.5 9.9v2L12 20.6H9.9Z"/>
  </svg>
  ラベル
</span>
```

上書き可能なカスタムプロパティは下記の通りです。

| プロパティ | 用途 |
| --- | --- |
| `--_non-text` | `outline`と`filled-outline`タイプの外枠、`fill`タイプの背景、アイコン |
| `--_bg` | `filled-outline`タイプの背景 |
| `--_text` | `text`と`outline`タイプのテキスト |
| `--_text-dark` | `filled-outline`タイプのテキスト |
