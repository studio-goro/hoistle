## 使い方

### 基本的な使い方

説明リストは`<dl>`、`<dt>`、`<dd>`要素を使用します。各項目を`<div>`要素で囲むことでグループ化できます。

```html
<dl class="dads-description-list">
  <div class="dads-description-list__item">
    <dt>項目名1</dt>
    <dd>これは項目1の説明文です。</dd>
  </div>
  <div class="dads-description-list__item">
    <dt>項目名2</dt>
    <dd>これは項目2の説明文です。</dd>
  </div>
</dl>
```

### ブレット付き

マーカーとしてブレット（丸点）を表示する場合は、`data-marker="bullet"`を指定します。

```html
<dl class="dads-description-list" data-marker="bullet">
  <div class="dads-description-list__item">
    <dt>項目名1</dt>
    <dd>これは項目1の説明文です。</dd>
  </div>
  <div class="dads-description-list__item">
    <dt>項目名2</dt>
    <dd>これは項目2の説明文です。</dd>
  </div>
</dl>
```

### カスタムマーカー付き

独自のマーカー（連番、アイコンなど）を使用する場合は、`data-marker="custom"`を指定します。カスタムマーカーは`<dt>`要素の最初の`<span>`要素で指定します。

```html
<dl class="dads-description-list" data-marker="custom">
  <div class="dads-description-list__item">
    <dt><span>1.</span>項目名1</dt>
    <dd>これは項目1の説明文です。</dd>
  </div>
  <div class="dads-description-list__item">
    <dt><span>2.</span>項目名2</dt>
    <dd>これは項目2の説明文です。</dd>
  </div>
</dl>
```

アイコンをマーカーとして使用することもできます。

```html
<dl class="dads-description-list" data-marker="custom">
  <div class="dads-description-list__item">
    <dt>
      <span><img src="icon.svg" alt="①" width="24" height="24" style="position: relative; bottom: -5px; margin-top: -5px; display: block;"></span>
      項目名1
    </dt>
    <dd>これは項目1の説明文です。</dd>
  </div>
</dl>
```
