## 仕様

### バリエーション

| 属性 | 要素 | 値 | 説明 |
| --- | --- | --- | --- |
| `data-size` | opener | `sm`, `md` | オープナーボタンのサイズ |
| `data-style` | opener | `text`, `outlined`, `filled` | オープナーボタンのスタイル |
| `data-text-weight` | opener | `normal`, `bold` | オープナーボタンのフォントウェイト |

### 機能仕様

#### カスタム要素

`<dads-menu-list-box>`はカスタム要素として実装されており、以下の機能を提供します。

- メニューの開閉
- キーボードナビゲーション
- フォーカス管理
- 外部クリックでの閉じ処理

#### キーボード操作

| キー | オープナーにフォーカス時 | メニュー項目にフォーカス時 |
| --- | --- | --- |
| `Enter` / `Space` | メニューを開閉 | 項目を選択 |
| `↓` | メニューを開き最初の項目にフォーカス | 次の項目にフォーカス |
| `↑` | メニューを開き最後の項目にフォーカス | 前の項目にフォーカス |
| `Home` | — | 最初の項目にフォーカス |
| `End` | — | 最後の項目にフォーカス |
| `Escape` | — | メニューを閉じてオープナーにフォーカス |

#### カスタムイベント

メニュー項目が選択されると`menuitemselect`イベントが発火します。

```javascript
menuListBox.addEventListener('menuitemselect', (event) => {
  console.log(event.detail.selectedItem);   // 選択されたDOM要素
  console.log(event.detail.selectedValue);  // 選択された項目のテキスト
  console.log(event.detail.selectedIndex);  // 選択された項目のインデックス
});
```

## 使い方

### インストール

#### 必要なファイル

- **コンポーネント本体**
  - `menu-list-box.css`
  - `menu-list-box.js`
- **依存ファイル**
  - `menu-list.css`

スクリプトを読み込む際は必ず、`<script type="module">`を使用しES Moduleとして読み込んでください。

```html
<link rel="stylesheet" href="menu-list.css">
<link rel="stylesheet" href="menu-list-box.css">
<script type="module" src="menu-list-box.js"></script>
```

### マークアップ

#### 必須のARIA属性

アクセシビリティを確保するため、以下の属性を正しく設定してください。

##### オープナーボタン

- `id`: メニューの`aria-labelledby`から参照される
- `aria-controls`: ポップアップメニューのIDを指定
- `aria-haspopup="menu"`: メニューを持つことを示す
- `aria-expanded`: メニューの開閉状態（初期値は`"false"`）

##### ポップアップメニュー

- `role="menu"`: メニューであることを示す
- `aria-labelledby`: オープナーボタンのIDを参照

##### メニュー項目

- `role="menuitem"`: メニュー項目であることを示す
- 親の`<li>`には`role="presentation"`を設定

```html
<button id="opener-id" aria-controls="menu-id" aria-haspopup="menu" aria-expanded="false" ...>
  ...
</button>
<div class="dads-menu-list-box__popup" hidden>
  <ul id="menu-id" role="menu" aria-labelledby="opener-id">
    <li role="presentation">
      <button role="menuitem" ...>...</button>
    </li>
  </ul>
</div>
```

#### メニュー項目のスタイル

ポップアップ内のメニュー項目には`data-type="box"`を使用してください。`data-type="standard"`は角丸のスタイルで、サイドバーナビゲーションなどに適しています。

```html
<button class="dads-menu-list__item" data-type="box" data-size="regular" role="menuitem">
  ...
</button>
```
