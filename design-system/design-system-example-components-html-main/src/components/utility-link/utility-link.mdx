## 使い方

### アイコンの配置

リードアイコン（`dads-utility-link__lead-icon`）はリンクの内容を視覚的に補足するための装飾です。必要に応じて追加・省略できます。

```html
<!-- アイコンなし -->
<a class="dads-utility-link" href="#">
  <span class="dads-utility-link__label">リンクテキスト</span>
</a>

<!-- アイコンあり -->
<a class="dads-utility-link" href="#">
  <svg class="dads-utility-link__lead-icon" ... aria-hidden="true">...</svg>
  <span class="dads-utility-link__label">リンクテキスト</span>
</a>
```

### 外部リンクの実装

外部リンク（`target="_blank"`）の場合、末尾に外部リンクアイコン（`dads-utility-link__tail-icon`）を追加します。アイコンには`role="img"`と`aria-label="新規タブで開きます"`を設定し、スクリーンリーダーでリンクの動作を伝えます。

```html
<a class="dads-utility-link" href="#" target="_blank">
  <svg class="dads-utility-link__lead-icon" width="16" height="16" viewBox="0 0 24 24" fill="currentcolor" aria-hidden="true">
    <path d="..."/>
  </svg>
  <span class="dads-utility-link__label">リンクテキスト</span>
  <svg class="dads-utility-link__tail-icon" width="16" height="16" viewBox="0 0 48 48" fill="currentcolor" role="img" aria-label="新規タブで開きます">
    <path d="M22 6V9H9V39H39V26H42V42H6V6H22ZM42 6V20H39V11.2L21 29L19 27L36.8 9H28V6H42Z"/>
  </svg>
</a>
```
