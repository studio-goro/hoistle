## 仕様

### バリエーション

| 属性 | 値 | 説明 |
| --- | --- | --- |
| `data-type` | `stacked`, `inlined`, `stacked-underlay` | レイアウトタイプ。`stacked`はインジケーターとラベルを縦並び、`inlined`は横並び、`stacked-underlay`は縦並び+背景パネル付き |
| `intent` | `explicit`, `passive` | スクリーンリーダーへの通知方法。`explicit`はユーザーの明示的な操作で起動する場合（通知あり）、`passive`はスクロール等の暗黙的な操作で起動する場合（通知なし） |
| `value` | `0`〜`100`の数値 | 進捗率。指定すると確定進捗（Fill）モード、省略すると不確定進捗（Loop）モード |
| `announce-interval` | 秒単位の正の数値 | スクリーンリーダーへの進捗通知間隔（秒）。省略時は`5`。`intent="explicit"`のときのみ有効 |

### 機能仕様

#### 進捗モード

- **Loop（不確定進捗）**: `value`属性を省略した場合。処理時間が不明な場合に使用します。インジケーターがループアニメーションを表示します。
- **Fill（確定進捗）**: `value`属性を指定した場合。進捗率が把握できる場合に使用します。`value`の値に応じてインジケーターが充填されます。パーセンテージ表示用の`dads-progress-indicator__percentage`要素がある場合、自動的に進捗率が表示されます。

#### スクリーンリーダー通知（`intent="explicit"`の場合）

`intent="explicit"`が設定されている場合、以下のタイミングでスクリーンリーダーに通知されます。

| タイミング | 通知内容 |
| --- | --- |
| ローディング開始時 | 「読み込みを開始しました」 |
| 開始から`announce-interval`秒後 | 不確定進捗: 「読み込み中です」 / 確定進捗: 「\{value\}% 読み込みました。」 |
| 以降`announce-interval`秒ごと | 同上（繰り返し通知） |
| ローディング完了時 | 「読み込みが完了しました」 |

#### `prefers-reduced-motion`

`prefers-reduced-motion: reduce`が有効な環境では、スピナーおよびリニアのアニメーションが停止されます。

## 使い方

### インストール

#### 必要なファイル

- **CSS**
  - `progress-indicator.css`
- **JavaScript**
  - `progress-indicator.js`

スクリプトを読み込む際は必ず、`<script type="module">`を使用しES Moduleとして読み込んでください。

### 読み込みを開始・停止する

`active`属性がついているときにローディング状態が表示され、ない場合は非表示になります。

ページ読み込み時からローディングを開始したい場合は、HTMLに`active`属性を直接記述します。

```html
<dads-progress-indicator class="dads-progress-indicator" data-type="stacked" intent="passive" active>
  <!-- ... -->
</dads-progress-indicator>
```

動的にローディングを開始・停止する場合は、JavaScriptで制御します。

```javascript
const progress = document.querySelector('dads-progress-indicator');

// ローディング開始
progress.start();

// ローディング停止
progress.stop();
```

### `intent`属性の選び方

`intent`属性は必須です。コンポーネントの起動方法に応じて適切な値を選択してください。

- **`explicit`**: ユーザーがボタンを押す、フォームを送信するなど、明示的な操作でローディングが開始される場合に使用します。スクリーンリーダーへの開始・完了・進捗の通知が行われます。
- **`passive`**: 無限スクロール、画面遷移、バックグラウンド処理など、ユーザーの暗黙的な操作や自動的にローディングが開始される場合に使用します。スクリーンリーダーへの通知は行われません。

### 確定進捗でパーセンテージを表示する

確定進捗（Fill）モードでパーセンテージを表示するには、ラベル内に`dads-progress-indicator__percentage`要素を含めます。JavaScriptが進捗率に応じて自動的にパーセンテージを更新します。

```html
<span class="dads-progress-indicator__label" data-js-label>読み込み中 <span class="dads-progress-indicator__percentage" data-js-percentage></span></span>
```

### JavaScript API

コンポーネントはJavaScript経由で操作します。

```javascript
const progress = document.querySelector('dads-progress-indicator');

// ローディング開始
progress.start();

// ローディング停止
progress.stop();

// 進捗率の設定（0〜100）
progress.value = 50;

// 進捗率の取得
console.log(progress.value); // 50

// アクティブ状態の確認
console.log(progress.active); // true
```

| メソッド / プロパティ | 説明 |
| --- | --- |
| `start()` | ローディングを開始します。`active`属性が付与され、コンポーネントが表示されます。 |
| `stop()` | ローディングを停止します。`active`属性が削除され、コンポーネントが非表示になります。 |
| `value` | 現在の進捗率を取得・設定します。`null`の場合は不確定進捗モードです。 |
| `active` | ローディング中かどうかを取得します。 |
| `intent` | `intent`属性の値を取得します（`"explicit"` / `"passive"`）。 |

## カスタマイズ

### 通知メッセージのカスタマイズ

スクリーンリーダーへの通知メッセージは`data-announce-*`属性でカスタマイズできます。

```html
<dads-progress-indicator
  class="dads-progress-indicator"
  data-type="stacked"
  intent="explicit"
  data-announce-start="Loading started"
  data-announce-end="Loading complete"
  data-announce-long="Still loading"
  data-announce-long-with-value="{value}% loaded"
>
  <!-- 内容は省略 -->
</dads-progress-indicator>
```

| 属性 | デフォルトメッセージ | 利用可能な変数 |
| --- | --- | --- |
| `data-announce-start` | 読み込みを開始しました | なし |
| `data-announce-end` | 読み込みが完了しました | なし |
| `data-announce-long` | 読み込み中です | なし |
| `data-announce-long-with-value` | \{value\}% 読み込みました。 | `{value}` |

### 通知間隔のカスタマイズ

`announce-interval`属性で、読み込み開始後の最初の通知タイミングおよびその後の繰り返し間隔を秒単位で指定できます。省略した場合は`5`秒です。

```html
<dads-progress-indicator
  class="dads-progress-indicator"
  data-type="stacked"
  intent="explicit"
  announce-interval="10"
>
  <!-- 内容は省略 -->
</dads-progress-indicator>
```
