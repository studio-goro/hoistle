## ステージ詳細

---

### ステージ1：探索

**タッチポイント**
- SNS（TikTok / Instagram 短尺動画）
- メディア

**ユーザー行動**
1. SNSやメディアでの露出により、「ホイッスル」という名前をすでに知る
2. 休日や退勤後に暇を感じるが何もせずInstagramを閲覧している
3. フィードでホイッスルを見かける
4. 流れてきた瞬間に「見たことある・知ってる」「よく見るし人気そう」という印象を持つ

**ユーザー心理**

最初に見た時
- ポジティブ：なんとなく知っている存在として記憶に残っている。自分もスポーツを学生時にしていたな、と思い出す
- ネガティブ：知っているだけで、まだ関心は強くない

フィードでホイッスルを見かける
- ポジティブ：よく見かけるので気になっていた。何度か見ているので少し信頼できそうと感じる
- ネガティブ：人気そうでも自分に合うかはまだ分からない。認知はあるが、まだ行動する理由にはなっていない

---

### ステージ2：認知・興味

**タッチポイント**
- ホイッスルのInstagram（投稿・ストーリー・ハイライト）
- ホイッスルのプロフィール（リンク導線）
- Web検索

**ユーザー行動**
1. Instagramでホイッスルのアカウントにいき投稿を見る、もしくはそのままweb検索をして公式サイトに行く
2. 「信頼できる・ちゃんとしている」と感じる
3. 「普通の社会人サークルではない」なく「ヨガやジムのような大人向けのアクティビティ」と感じる
4. 社会人サークル＝参加費数百円という前提を持っている
5. 社会人サークルだと思ってホイッスルを見ると高いと感じる
6. ちゃんとサービスで、良い体験を受けれるから金額への納得感を感じてほしい
7. ヨガ・ジムは自己投資、そのように分類されたり、ちょっと自分へいいことしている、アクティビティ要素もある
8. 参加イメージが湧き、「一度行ってみたい」と思う

**ユーザー心理**

投稿を閲覧
- ポジティブ：ちゃんと運営されていそうで安心する。初めてでも参加しやすそう。健康的で前向きな時間が過ごせそう。一回くらいなら気軽に試してみたい
- ネガティブ：自分でも馴染めるのか少し気になる。興味はあるけど、詳細がわからなくて申し込むのはまだ少し迷う

---

### ステージ3：検討

**タッチポイント**
- 公式サイト（TOP / イベント詳細）

**ユーザー行動**
1. 参加の詳細を知りたいと思いSNSのプロフィールor検索をする
2. サイトに行くとInstagramにはない豊富な情報があり楽しそう・ハードルが低そうで参加したい気持ちが湧く
3. UGC（ユーザー投稿）を見る
4. 固定動画や導線で、サービス内容や参加方法がすぐ理解できる
5. 未経験向けのコンテンツもあり参加ハードルが下がる
6. 自分にあったイベントを探す
7. 日程・場所・カテゴリが整理された状態で表示される
8. イベントごとの違いや魅力が分かりやすく、比較しやすい
9. さらに会員登録をしてより自分に合うイベントがすぐ見つかる

**ユーザー心理**

公式サイトを閲覧する
- ポジティブ：楽しそうで一度行ってみたい気持ちが強くなる。初心者でも大丈夫そうで安心する。イベントが多くても自分に合うものが絞られて選びやすい
- ネガティブ：条件は合っても内容の違いがまだ分かりにくいかも。登録しないとできない操作があるのは少し面倒

---

### ステージ4：初回予約

**タッチポイント**
- 公式サイト（予約／決済画面）

**ユーザー行動**
- 参加したいイベントが決まり、予約ボタンをクリック
- 内容を軽く確認し、そのまま予約に進む
- 必要情報を入力し、決済・本人確認を行う
- 予約を完了する

**ユーザー心理**

イベントを予約する
- ポジティブ：スムーズに予約できそうで安心。参加するのが楽しみになってきた
- ネガティブ：入力が面倒に感じる。決済や個人情報の入力に少し不安がある。当日うまく参加できるか少し不安が残る

---

### ステージ5：参加前日まで（初回）

**タッチポイント**
- 運営からの連絡（通知／メッセージ／リマインド）

**ユーザー行動**
- 日常生活に戻り、予約したことを一旦忘れる
- 開催日が近づき、運営からの連絡を見る
- 「そろそろ開催日」「こんなことをする」「こんな人が来る」などの連絡を受け取る
- 参加のイメージが少し具体的になる
- 前日に運営から連絡が来る
- 場所・持ち物・時間・アクセスなど参加の基本情報と「こんなことをする」「こんな人が来る」をまとめた情報が届く
- 当日に向けて気持ちが上がる
- 「明日が楽しみだ」と少し楽しみな予定として意識される

**ユーザー心理**

開催日が近づき、運営からの連絡を見る
- ポジティブ：そういえば予約していたと思い出す。どんな雰囲気かイメージできて楽しみになる
- ネガティブ：急に思い出して少し面倒に感じる。今の予定との兼ね合いが気になる

前日に運営から連絡が来る
- ポジティブ：ちゃんとリマインドしてくれて助かる。必要な情報がまとまっていて分かりやすい。少しずつ楽しみな気持ちが出てくる
- ネガティブ：楽しみと同時に少し緊張する

### ステージ6：初回参加

**タッチポイント**
- 開催日までの通知（メール・LINEなど）
- 現地会場（体育館）
- 運営スタッフ

**ユーザー行動**
- 体育館に向かう
- 会場の場所や入口を探しながら到着する
- 受付でゼッケンを受け取る
- アプリで状況を確認し、スムーズにプレーをする
- チーム分け・進行が行われる
- 試合の合間に、近くの参加者や同じチームの人と話す
- プレーを通じて少しずつ場に慣れていく
- 終了後、全員でモップ掛けなど片付けをする
- 気が合った参加者ができる
- 次回開催への参加の約束をする
- 友達申請をし、連絡が取れる状態にする
- 解散して帰宅する

**ユーザー心理**

現地に向かう
- ポジティブ：少しの緊張はあるが、参加の楽しみが強い
- ネガティブ：場違いにならないか少し緊張する。道中あまり人がいなく、不安になる

到着する
- ポジティブ：ちゃんと参加者として受け入れられて安心。これから始まる感じがしてワクワクする
- ネガティブ：初対面で少し緊張する。どう振る舞えばいいか少し迷う

アプリで状況を確認し、スムーズにプレーをする
- ポジティブ：スムーズに進んでやりやすい。運営がしっかりリードしてくれるので安心
- ネガティブ：周りについていけているか気になる。自由度が少ないと感じる可能性もある

参加者との交流
- ポジティブ：少しずつ打ち解けられて楽しい。自然に会話できて安心する
- ネガティブ：知らない人と話すことは緊張する

片づけ
- ポジティブ：みんなでやることで距離が縮まる。懐かしい雰囲気になる
- ネガティブ：スポーツがしたいので、掃除は面倒

気が合った参加者ができる
- ポジティブ：いい人と出会えて嬉しい。次も参加する楽しみができる。今後も気軽に連絡が取れそう
- ネガティブ：断りづらい雰囲気だと困る

解散して帰宅
- ポジティブ：久しぶりにスポーツができてよかったと感じる。次回も参加したいと思う
- ネガティブ：少し気疲れしたと感じる。次も行くかは少し考えたい

### ステージ7：初回参加後

**タッチポイント**
- 運営からのフォロー連絡（アプリ／通知／メッセージ）

**ユーザー行動**
- 帰り道に運営から連絡が来る
- 今日のプレーの評価・動画が届き、閲覧し、自然と今日のことを振り返る
- 運営からのアンケートに回答する（参加者同士の評価）
- また参加したい、という気持ちが湧いてくる
- Instagramに今日のことをストーリーに載せる
- ストーリーを見た友達から「気になる」という反応をもらい、誘う
- 事前に友達を誘うとインセンティブが入ることを知っている状態
- 自分もまた参加したい気持ちだった

**ユーザー心理**

運営から連絡が来る
- ポジティブ：自分のプレーを振り返れて面白い。流れでそのまま予約できて楽
- ネガティブ：上手くできていなかった部分が気になる。勢いで予約して後で都合が合わないかもと不安になる

### ステージ8：2回目参加検討

- 「友だちが参加しています！」という情報が含まれる
- 自分にあったイベントのおすすめで配信される
- 気になってイベント詳細を開く
- 自分に合っているイベントかつ、参加日程も豊富だったので参加する

**ユーザー心理**

通知を閲覧する
- ポジティブ：自分に合いそうなイベントがあり興味が湧く。知り合いが参加しているなら楽しそう。スムーズに次の予定を決められて満足
- ネガティブ：通知が多いと少し煩わしく感じる

予約をする
- ポジティブ：スムーズに予約できてストレスがない

自身もスタッフとして手伝うようになる
- ポジティブ：自分も場づくりに貢献できていると感じる
- ネガティブ：責任が増えることに少しプレッシャーを感じる

## 画面作成時の参照ポイント

### デジタル画面が登場するステージ

| ステージ | 画面 | ユーザーの状態 |
|---|---|---|
| 3：検討 | 公式サイト TOP / イベント一覧 / イベント詳細 | 「行きたい気持ちはあるが、まだ迷っている」。情報量と安心感が決め手 |
| 4：初回予約 | 予約画面 / 決済画面 | 「決めた。でも入力は面倒」。摩擦を最小化することが重要 |
| 7：初回参加後 | フォロー通知 / 振り返り / 次回予約導線 | 「楽しかった余韻がある」。この勢いで次の予約に誘導する |
| 8：2回目検討 | 通知 / おすすめイベント | 「通知を見て思い出す」。自分ごと化された情報が重要 |
| 9：2回目予約 | 予約画面 | 「前回と同じ流れで安心してできる」。前回情報の引き継ぎが効果的 |
| 13：定着 | マイページ / スタッツ / バッジ | 「続けている実感がほしい」。成長・記録の可視化が継続を支える |

### 全ステージを通じたユーザーの感情的なテーマ

- **不安と安心の繰り返し**：各ステージで「ちゃんとしているか・自分に合うか・馴染めるか」という不安が生じ、情報や体験によって解消されていく
- **社会人サークルではなく自己投資サービスとしての認識**：価格への納得感を得るためには「ヨガ・ジムのような自己投資」という文脈づけが重要
- **人とのつながりが継続の鍵**：初回参加から2回目・定着へとつながる動機の中心は「また会いたい人ができた」という体験
- **運営への信頼が行動を後押しする**：「ちゃんとしている・親切」という印象が、予約・参加・継続の各判断に影響している
